String Data Analysis

Inspiration

I'm discovering about program analysis, with that comes data analysis and with simple examples of data analysis, comes boundary value analysis.

Boundary value analysis is a technique to have for each instance of a variable in a program, a par of values, the smallest and the biggest possible value of that variable at that instance.

There exists also the analysis to get all the values modulo a number.

That made me think if it existed also other simple properties for other types. for integers, floats, chars, etc it's pretty obvious, but one that made me really curious was string, because we have a good way to represent alot of variety in strings, it's called regular expressions
